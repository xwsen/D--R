Spiral staircase. บันไดวน.


งานออกเเบบเสารองรับบันไดวนสูง จุดประสงค์ในการออกเเบบ คือ หาขนาดของเสาโดยมีบันไดวนเป็นวงกลม รัศมีเท่ากัน มีระยะในแต่ละขั้นของบันไดเท่ากัน จากด้านบนสุดจนถึงล่างสุด เพื่อดูว่าเสาเหล็กวงกลมตันสามารถรับน้ำหนักกดสูงสุดได้กี่ kg วิธีคิดคำนวณการรับน้ำหนักของเสา - คำนวณหาพื้นที่หน้าตัด-ขนาดของเสาเหล็กรูปพรรณ วงกลมตัน P = เเรงกดที่กระทำบนเสา = 25 x 60 x 1.25 = 1,875 kg ( 25 -> จำนวนขั้นบันได, 60 -> น้ำหนักเฉลี่ยของคนบนบันได, 1.25 -> ค่าความปลอดภัยของการรับน้ำหนักของเสา 25 % ) A = พื้นที่หน้าตัดของเสา Allowble Stress ของเหล็กรูปพรรณ = 0.40 fy (fy=2400 ksc.) = 0.40x2400 = 960 kg./sq.cm (P/A) = 960 A=1,875/960 =1.953125 sq.cm เลือกใช้เหล็กรูปพรรณวงกลมตันขนาด = √1.953125 ≈ 1.4 cm ดังนั้นใช้เหล็กวงกลมตันขนาด 1.5x1.5 cm A = 2.25 sq.cm >1.953125 sq.cm สรุปเสาเหล็กกลมตันขนาด A = 2.25 sq.cm.รับน้ำหนักกดสูงสุด = 2.25x960 = 2160 kg
